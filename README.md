> Get a simple timestamp

Fast as it doesn't have bloated methods.


## Install

```sh
$ npm install --save simple-timestamp
```


## Usage

```js
var simpleTimestamp = require('simple-timestamp');

simpleTimestamp();
// => 2015.05.20.14.34.08
```

```sh
$ npm install --global simple-timestamp
$ simple-timestamp
2015.05.20.14.36.09
```
